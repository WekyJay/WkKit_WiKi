# Features

WkKit provides a comprehensive set of features for managing kits and enhancing player experience.

## Core Features

### Kit System
The foundation of WkKit - create, manage, and distribute item kits to players.

- **Multiple Kit Types**: Regular kits, scheduled kits, permission-based kits
- **Flexible Distribution**: Direct give, mail system, CDK redemption
- **Cooldown Management**: Prevent abuse with configurable cooldowns
- **Usage Limits**: Control how many times players can claim each kit

[Learn more about Kit System →](./kit-system.md)

### CDK System
Redemption code system for marketing campaigns and events.

- **Batch Generation**: Create multiple CDKs at once
- **Usage Tracking**: Monitor redemption statistics
- **Expiration Control**: Set time-limited codes
- **Custom Rewards**: Link CDKs to specific kits

[Learn more about CDK System →](./cdk-system.md)

## Additional Features

- **Kit Mail**: Send kits to offline players
- **Kit Reminders**: Notify players when cooldowns expire
- **Menu System**: Visual kit selection interface
- **PlaceholderAPI**: Full placeholder support
- **Multi-language**: Support for multiple languages
