## 🆕 新版命令特性详解

### 1. `/wk info <礼包名称>`
**功能**: 查看礼包详细信息并打开预览界面
**新版特性**:
- 显示礼包完整配置信息
- 统计物品数量、命令数量、冷却时间等
- 自动打开物品预览界面
- 权限、冷却、使用次数状态检查

### 2. `/wk create <礼包名称>`
**功能**: 创建新礼包
**新版特性**:
- 智能默认配置生成
- 自动使用手中物品作为图标
- 完整的配置验证
- 引导后续编辑命令

### 3. `/wk give <玩家> [礼包名称]`
**功能**: 给玩家发放礼包
**新版特性**:
- 完整验证: 权限、冷却、使用次数、背包空间
- 异步发放: 不阻塞服务器主线程
- 详细反馈: 成功/失败原因
- 列表模式: 不指定礼包时显示玩家可领取列表

### 4. `/wk reload`
**功能**: 重载插件配置
**新版特性**:
- 热重载: 不影响运行中服务
- 配置验证: 重载前检查配置有效性
- 状态保持: 玩家数据不受影响

## 📋 使用示例

### 示例1: 查看礼包信息
```bash
/wk info 新手礼包
```
显示礼包详情，统计信息，并打开物品预览界面

### 示例2: 创建新礼包
```bash
/wk create 活动礼包
```
创建名为"活动礼包"的新礼包，使用手中物品作为图标

### 示例3: 给玩家发放礼包
```bash
# 显示玩家可领取礼包列表
/wk give Steve

# 发放指定礼包
/wk give Steve 新手礼包
```

### 示例4: 查看命令帮助
```bash
/wk help
```
显示所有可用命令及详细用法
