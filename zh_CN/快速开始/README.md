# 🚀 WkKit 快速开始指南

欢迎使用 WkKit！本指南将帮助您快速安装和配置插件，让您在几分钟内开始使用礼包系统。

## 📦 安装步骤

### 第1步：下载插件
1. 访问 [WkKit 发布页面](https://github.com/WekyJay/WkKit/releases)
2. 下载最新版本的 `WkKit.jar` 文件
3. 确保服务器运行 **Spigot 1.8+** 或 **Paper 1.8+**

### 第2步：安装插件
1. 将 `WkKit.jar` 放入服务器的 `plugins/` 目录
2. 重启服务器或使用 `/reload confirm` 命令
3. 等待插件生成配置文件

### 第3步：验证安装
```bash
# 在游戏内执行
/wk help

# 或在控制台执行
help wk
```
如果看到命令列表，说明安装成功！

## ⚙️ 基本配置

### 1. 首次运行配置
插件首次运行会自动生成：
```
plugins/WkKit/
├── config.yml          # 主配置文件
├── messages.yml        # 消息配置文件
├── kits/              # 礼包配置目录
│   └── example.yml    # 示例礼包
└── menus/             # 菜单配置目录
    └── example.yml    # 示例菜单
```

### 2. 修改主配置
编辑 `plugins/WkKit/config.yml`：
```yaml
# 数据库配置（默认使用SQLite）
database:
  type: sqlite
  file: plugins/WkKit/data.db
  
# 语言设置
language: zh_CN

# 更新检查
update-check: true
```

### 3. 创建第一个礼包
1. 在游戏内手持想要设置为礼包图标的物品
2. 执行命令：
```bash
/wk create 新手礼包 "欢迎礼包" default
```
3. 打开礼包编辑界面配置内容

## 🎮 快速体验

### 体验流程
1. **创建礼包** → 2. **配置内容** → 3. **设置权限** → 4. **测试发放**

### 示例：创建每日登录礼包
```bash
# 1. 创建礼包
/wk create 每日登录 "每日登录奖励" daily

# 2. 打开编辑界面
/wk edit 每日登录

# 3. 在界面中添加物品
#    - 点击物品槽位添加物品
#    - 设置经济奖励
#    - 配置命令执行

# 4. 设置冷却时间（24小时）
#    在配置文件中设置 delay: 86400

# 5. 测试礼包
/wk get 每日登录
```

## 🔧 常用命令速查

| 命令 | 说明 | 权限 |
|------|------|------|
| `/wk help` | 查看帮助 | 所有玩家 |
| `/wk create <名称>` | 创建礼包 | wkkit.create |
| `/wk edit <名称>` | 编辑礼包 | wkkit.edit |
| `/wk kits` | 查看所有礼包 | wkkit.list |
| `/wk get <名称>` | 领取礼包 | wkkit.get |
| `/wk give <玩家> <礼包>` | 发放礼包 | wkkit.give |

## ❓ 遇到问题？

### 常见问题解决
1. **命令不生效** → 检查权限和命令拼写
2. **礼包无法领取** → 检查冷却时间和权限
3. **物品丢失** → 检查背包空间和物品配置
4. **配置错误** → 检查YAML语法和缩进

### 获取帮助
- **查看详细文档**：本Wiki的所有章节
- **查看游戏内帮助**：`/wk help`
- **检查配置文件**：参考配置示例
- **联系开发者**：GitHub Issues
